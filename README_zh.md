# 🧠 Product Manager Copilot | 资深产品经理助手

*一位资深 PM Copilot，擅长通过结构化的“七步讨论法”将模糊需求转化为高标准产品交付物。*

Product Manager Copilot 是你工作中的最佳智囊，专门负责将一句话的产品想法转化为具有落地价值的《标准需求卡片》与《PRD》。它严格遵循行业标准的“七步讨论法”，带领你从战略定位到战术拆解，层层剖析业务痛点，确保每一个需求都有清晰的闭环。

---

### ✨ 核心亮点

-   **🪜 七步讨论法**：从定义问题、理想态到具体方案，通过循序渐进的结构化讨论，避免需求分析时的逻辑遗漏。
-   **📋 标准需求卡片**：在立项阶段帮助你明确关键约束与核心 Gap，确保团队在投入资源前“做正确的事”。
-   **📝 高标准 PRD 产出**：不仅整理详实的业务规则和边界，更能自动生成包含 Mermaid 语法的标准业务流程图。
-   **🤖 AI 特性把控**：专门针对大模型和概率性功能设置校验点，提醒补充实验数据与兜底策略，降低上线风险。

---

### 🚀 快速开始

想要将一个模糊的想法转化为严谨的产品需求文档，只需在对话框中输入以下指令：

> **"请启用 Product Manager Copilot 技能。我有一个产品想法，想为我们的 AI 问答产品增加多轮对话功能，帮我把这个需求想清楚。"**

# 📋 product-manager-copilot | 资深产品经理助理

*你的专属高阶 PM Copilot，将模糊的业务想法转化为研发直接可用的高标准 PRD。*

product-manager-copilot 是一个结构化的需求分析与 PRD 撰写技能。它通过“七步讨论法”，引导用户从业务痛点出发，经过战略推演、AI 实验校验、路线权衡，最终产出符合工程标准的《需求卡片》与《产品需求文档》。它能有效避免需求定义不清、业务规则与技术细节混淆等常见产品踩坑点。

---

### ✨ 核心亮点

-   **🪜 结构化七步法**：基于“战略到执行”的完整推演逻辑，从问题定义到最终 DoD（验收标准）单步启发，避免信息过载。
-   **🛑 动态 AI 校验点**：专为 AI 概率性需求设计，在决策前强制引入实验测试回路，用真实数据而非直觉指导技术路线。
-   **🛡️ 智能风险规避**：内置“模块划分四问”与历史避坑记录（Lessons Learned）读取机制，自动纠正混淆业务规则与技术细节的坏味道。
-   **📐 标准化物料产出**：输出带有 Mermaid 流程图的标准化 PRD 与需求卡片，并在私有工作区中长效管理公司上下文背景。

---

### 🚀 快速开始

想要快速理清需求并输出标准文档，只需在对话框中输入以下指令：

> **"请启用 product-manager-copilot 技能。我有一个新产品想法：我们要为 AI 问答增加多轮对话功能，帮我梳理一下思路并产出 PRD。"**
